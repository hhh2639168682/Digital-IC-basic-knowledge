# 列出文件
**ls** 列出当前目录下的文件  
ls -f 取消排序  
ls -F 区分文件和文件夹  
ls -a 显示所有文件，包括隐藏文件  
ls -l 长列表显示，显示更多文件信息  
# 操作文件
**touch filename** 创建文件  
**cp test documents/** 复制test到document文件夹  
cp /etc/network/*.conf ./ 复制.conf到当前文件夹  
cp -r(-R) scripts/ mod_scripts 复制scripts下所有内容到mod_scripts下  
**ln -s** 软链接
例如 ln -s /mnt/big_drive/waveforms /mnt/small_drive/simulation/waveforms
